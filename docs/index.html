About Matui

Matui is the browser with material design user interface. It is focused on simplicity and built for the typical user. Not many features yet but definitely looks cool:
  • Much minimalistic.
  • Kinda lightweight.
  • Very chromium
  • It works-ish (still in developement)
Here are some planned features though:
  • Built in adblock/tracker prevention so users can have faster load times and more privacy protection
  • Interface customization with colors and quick access buttons
  • Definitely needs keyboard shortchuts and custom keybindings
