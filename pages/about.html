About Matui

Matui - The material user interface browser
Version: 0.2.23
Check for updates
The goal of this project is to provide an uncluttered browsing experience while keeping many of the powerful features in popular modern browsers. Matui uses minimalistic design principles that results in a simple to use and clean interface for surfing the web.
Matui uses Chromium and Node.js technologies and is built with Electron.
