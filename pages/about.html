About Matui

Matui Version: pre-release 0.1
Check for updates
Matui is your material user interface browser - a modern web browser created for simplicity.
The goal is to provide an uncluttered browsing experience while keeping the many powerful features of popular browsers. Using minimalistic design principles, Matui offers a clean interface for surfing the web. It also has built-in adblock and prevents trackers from identifying users, all to improve load times and protect privacy.
Matui uses Chromium and Node.js technologies and is built with Electron.
